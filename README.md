# Modular-Hits

A collection of one shots sampled from a Eurorack Modular synthesizer.

-  **Formats:** AIF & WAV
-   **Approx Size:** 94mb
- [Web Audio Demo](https://www.modularsamples.com/Demos/demos/modularhits.html)
